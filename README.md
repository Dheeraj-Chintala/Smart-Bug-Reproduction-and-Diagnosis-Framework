

## Setup

### 1. Clone repo
```bash
git clone https://github.com/Dheeraj-Chintala/Smart-Bug-Reproduction-and-Diagnosis-Framework.git
```

### 2. Add OpenRouter API Key in .env



VITE_OPENROUTER_API_KEY=your_openrouter_api_key_here




### 3. Install Dependencies
```bash
npm install
```


### 4. Run Development Server
```bash
npm run dev
```
The app will run at:

http://localhost:5173
